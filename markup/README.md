# Markup

This folder is currently a repository for examples of the future wiki's markup.

Once the basic data structure is finalized, clientside and serverside libraries for parsing the markup will also go here.

The idea is to create examples of the source documents that will be stored in a document store like mongoDB. The libraries will then be designed to convert the source documents to and from other languages.

### For example:
- Converting JSON source to HTML
- Converting Wetfish Wiki markup to JSON
- Converting MediaWiki markup to JSON
- Converting Markdown to JSON


By using a generic data structure, it lets us convert content from other markup formats, reducing the learning curve for new users.
